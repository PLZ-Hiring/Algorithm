# 💼 PLZ-Hiring

> "From *please hire me* to *you're hired* 🚀" 
> 취업 준비를 위한 꾸준한 알고리즘 스터디입니다.

---

## 🧠 스터디 개요

**PLZ-Hiring**은 격주로 진행되는 취업 대비 알고리즘 스터디입니다.  
각 주차별로 핵심 개념에 맞는 문제를 직접 풀이하고,  
스터디 시간에는 발표 및 토론을 통해 다양한 접근법을 공유합니다.

---

## 📅 진행 방식

### ✅ 일정
- **진행 주기:** 격주 금요일 **23시**
- **문제 풀이 마감:** 스터디 전날 **23:59**

### ✅ 구성
1. 각 주차별 개념과 관련된 **10문제**를 엄선합니다.  
2. 스터디 시작 시 **발표 순서**를 정합니다.  
3. 각자 원하는 문제를 선택해 **1~2문제씩 발표 및 토론**을 진행합니다.  
   - 단, **이미 발표된 문제는 중복 발표 불가!**  
4. 발표자는 자신의 풀이 및 접근 방법, 배운 점을 공유합니다.

---

## 💻 언어

- **프로그래밍 언어:** 자유 선택
